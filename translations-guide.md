## 文件說明

此文件為 VerifyAI landing page 的翻譯管理指南，協助文案人員理解並編輯各語言版本的內容。

**支援語言：**
- 🇹🇼 繁體中文 (zh-TW) - 主要語言
- 🇺🇸 英文 (en)
- 🇯🇵 日文 (ja)
- 🇰🇷 韓文 (ko)

**相關檔案：**
- `translations.js` - 翻譯資源檔（實際編輯此檔案）
- `verifyai-complete-final_4.html` - 主頁面檔案
- `translations-guide.md` - 本指南文件

## 如何編輯翻譯

### 步驟 1: 打開翻譯檔案
使用任何文字編輯器打開 `translations.js`

### 步驟 2: 找到要編輯的語言區塊
檔案分為四個主要區塊：
```javascript
'zh-TW': { ... }  // 繁體中文
'en': { ... }     // 英文
'ja': { ... }     // 日文
'ko': { ... }     // 韓文
```

### 步驟 3: 找到對應的翻譯鍵
使用本文件的「翻譯鍵值對照表」查找要修改的內容位置

### 步驟 4: 修改內容
```javascript
// 修改前
'hero_title_line1': '讓信任有依據',

// 修改後
'hero_title_line1': '讓信任更有依據',
```

### 步驟 5: 儲存並測試
- 儲存檔案
- 在瀏覽器中打開 `verifyai-complete-final_4.html`
- 測試語言切換功能是否正常

## 品牌語調指南

### 核心品牌訊息
**主標語：「讓信任有依據」**
- 英文：Trust Made Tangible
- 日文：信頼に根拠を
- 韓文：신뢰에 근거를

### 語調原則

#### 1. 專業而親切
- ✅ 使用「你」而非「您」（中文）
- ✅ 保持友善但不過度輕鬆
- ❌ 避免過於技術性的術語

#### 2. 重視安全與信任
- ✅ 強調「守護」、「保護」、「安心」
- ✅ 突出速度「3 秒」和規模「數十億」
- ❌ 不要過度承諾或誇大

#### 3. 行動導向
- ✅ 使用明確的 CTA：「開始使用」、「立即驗證」
- ✅ 簡潔有力的動詞
- ❌ 避免模糊或被動的表達

### 各語言特殊考量

#### 🇹🇼 繁體中文
- 使用台灣慣用詞彙（如「部落格」而非「博客」）
- 保持親切感，使用「你」而非「您」

#### 🇺🇸 英文
- 使用美式英語拼寫
- 保持簡潔，避免冗長句子
- 專業但易懂的用語

#### 🇯🇵 日文
- 使用敬體（です・ます體）
- 適當混用漢字和假名，提升可讀性
- 保持禮貌但不過於正式

#### 🇰🇷 韓文
- 使用標準韓文（現代口語）
- 適當混用漢字詞和固有詞
- 保持親切感

## 常見問題

### Q1: 修改翻譯後需要重新整理瀏覽器嗎？
**A:** 是的，因為 `translations.js` 會被瀏覽器快取。建議使用強制重新整理（Ctrl+F5 或 Cmd+Shift+R）。

### Q2: 可以只修改某一個語言嗎？
**A:** 可以，每個語言的翻譯都是獨立的，修改一個不會影響其他語言。

### Q3: 如何增加新的翻譯項目？
**A:** 需要同時修改：
1. `translations.js` - 在所有語言區塊中加入新的鍵值對
2. `verifyai-complete-final_4.html` - 在對應元素加入 `data-i18n="new_key"` 屬性

### Q4: emoji 表情符號需要翻譯嗎？
**A:** 不需要，emoji 是通用符號，建議所有語言保持一致。

### Q5: 數字（如「3 秒」、「01, 02, 03」）需要翻譯嗎？
**A:**
- 步驟編號 (01, 02, 03)：保持阿拉伯數字
- 描述中的數字（3 秒）：建議保持阿拉伯數字，但可根據當地習慣調整

### Q6: 品牌名稱「VerifyAI」需要翻譯嗎？
**A:** 不需要，品牌名稱在所有語言中保持 "VerifyAI"。

### Q7: 如何確保翻譯品質？
**A:** 建議：
- 實際在瀏覽器中查看效果
- 請母語者審核
- 確保符合品牌語調
- 檢查字數長度是否適合版面

### Q8: 翻譯文字太長導致版面跑版怎麼辦？
**A:**
1. 嘗試使用更簡潔的表達
2. 諮詢開發團隊調整 CSS 樣式
3. 在本指南中標註該項目需要長度限制
